本站说明About

本站是一套面向工科背景读者的 A 股财报与公告教学系统. 把会计学 / 公司财务 / 证券投资学三个领域用first principles + 工程类比 + A 股真实案例三层组织起来. 目标: 让一个数理基础好但金融会计零基础的读者, 在 ~40 小时学习后能独立分析任意 A 股年报与公告。

设计原则

  • 每个知识点三层呈现: 直觉 (类比) → 定义与公式 (含 LaTeX + 准则引用) → A 股真实案例 (公司 + 代码 + 数字)
  • 反直觉题驱动: 每章 3-5 道陷阱题, 答案折叠, 鼓励先想后看
  • 专家分歧透明: 每模块导览列出业界最激烈的分歧 (Fama vs Shiller, DCF vs 相对估值, MM 现实适用性等)
  • 三模块互引: 复习引用具体章节路径, 知识形成网状
  • 纯静态 + AI 问答: 无后端, 浏览器直连 Anthropic API, 投资者 API key 自管

技术架构

  • 纯 HTML + CSS + ES module JavaScript (无 React / Vue / 任何前端框架)
  • KaTeX 公式渲染 (auto-render)
  • AI 助手: Anthropic Claude Opus 4.7, 浏览器直连 api.anthropic.com
  • 本地启动: python3 -m http.server 8000
  • 样式: 学术 paper 风格 (Cormorant Garamond + Source Serif 4 + Noto Serif SC + JetBrains Mono)

建议学习路径

最短路径 (~ 12 小时)
M1: 01 → 02 → 08 → 10 (读财报)
M2: 03 → 08 → 10 (公司财务核心)
M3: 05 → 07 → 09 (估值与公告)
A 股: 04 (避雷)
完整系统 (~ 40 小时)
从 M1 Ch.00 顺序读到 A 股 Ch.05 + 全部检验题. 配合 Python 实操 (Pandas 处理 A 股财报数据)
查询用
把本站当作"参考手册" — 遇到具体问题翻对应章节. 配合 Glossary 跳转

使用 AI 助手

右下浮动按钮 → 输入 Anthropic API Key (sk-ant-...) → 开始对话. Key 仅存浏览器 localStorage, 不发送任何第三方. 推荐用法:

  • 遇到不理解的概念: "解释一下 [概念名], 用工程类比"
  • 遇到具体 A 股公司: "[公司代码] 的 [指标] 怎么看?"
  • 遇到检验题不会: "Module II Ch.10 Q1, 我的思路是 X, 对吗?"
  • 遇到具体年报: 描述财报数字, 请助手分析风险

免责声明

本站为教学用途. 所有 A 股案例的数字均为近似值, 实际投资决策必须查阅原始年报 (巨潮 cninfo). 案例中对公司经营状况、造假指控等表述基于公开信息, 不构成任何投资建议或诽谤. 投资有风险, 决策需谨慎。

致谢

本站内容综合参考了 Brealey, Bodie, Damodaran, Penman, Graham, Buffett, Munger 等经典著作, 以及 Fama, Markowitz, Modigliani, Miller, Sharpe, Sloan, Beneish, Altman, Piotroski, Myers, Jensen, Kahneman, Shiller 等学术先驱的开创性研究. 中国会计学者葛家澍、夏冬林等的中国 CAS 体系工作亦是本站理解中国特色的基础.
