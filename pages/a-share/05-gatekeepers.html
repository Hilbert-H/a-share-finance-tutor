中介机构与问询函

A 股投资者最有力的"外部专家意见": 审计师意见 + 交易所问询函. 这两个信号在暴雷案例中提前预警率 ~90%, 但大量散户忽视.

① 审计意见五类

类型含义市场反应
标准无保留 (清洁)报表真实、完整、合规—
带强调事项段的无保留总体清洁, 但有特别注意事项轻负面
保留意见个别事项不确认中度负面
无法表示意见审计师无法获取充分证据重大负面, 通常 ST
否定意见报表整体不真实退市风险

历史规律: 大量 A 股暴雷案在暴雷前一年已收到非标意见. 康美 2017 无法表示意见 → 2018 自爆; 康得新 2018 保留意见 → 2019 暴雷.

实操规则: 看到任何非标意见, 立即停止任何买入, 已持仓考虑卖出 (无论价格).

② 关键审计事项 (KAM)

2017 起 A 股审计报告必须列出 KAM — 审计师投入显著精力判断的领域. 不是"问题", 但是"难以判断之处". 反复出现同一 KAM = 系统性判断难点 → 潜在雷区.

典型 KAM: 商誉减值测试 / 收入确认 / 应收坏账 / 存货跌价 / 非上市股权 FV / 关联方实质 / 递延税资产可收回性.

③ 交易所问询函

2017 后, 交易所对年报普遍发事后审核问询函. 类型:

  • 年报问询函: 财报披露后, 交易所对疑问点要求解释 (最常见)
  • 定期报告问询函: 季报 / 半年报问询
  • 重大事项问询: 重组 / 收购 / 担保 / 大额诉讼等具体事件
  • 异常波动问询: 股价异常波动要求公司澄清

典型问询函内容: "请说明 XX 科目余额异常增长原因 / 关联方资金往来真实性 / 商誉减值假设合理性 / 应收账款回收风险". 公司必须 5-10 个交易日内回复, 中介机构同意回复内容.

问询函如何使用:

  1. 看公司是否密集(年内多次) 收到问询函 — 多则警惕
  2. 看问询函具体内容 — 询问的领域往往是真实问题
  3. 看公司回复质量 — 详细数据 + 中介机构背书 = 合规; 模糊 / 反复 / 失误 = 危险
  4. 看是否多轮问询 (交易所对回复不满意继续追问) — 极强信号

④ 监管函与处罚

问询函上一级 = 监管函 (实质性违规) → 行政处罚 (罚款 / 警告 / 限制) → 立案调查 (刑事可能) → 重大违法退市.

监管函公告即应立即重视 — 表示 CSRC 或交易所已认定实质问题, 不只是"建议补充披露".

⑤ "看门人"被处罚的趋势

康美案后, 中介机构责任大幅强化:

  • 正中珠江会计师事务所被罚 5,000 万 + 暂停证券业务资格
  • 多名签字 CPA 终身证券市场禁入
  • 保荐人若有问题, 保荐机构连带责任

选择审计师本身就是公司质量信号: "四大"(普华永道, 安永, 德勤, 毕马威) 或内资八大 (立信, 信永中和, 天健等) 审计的公司, 一般质量更高; 小事务所审计 + 大公司不匹配, 可疑.
