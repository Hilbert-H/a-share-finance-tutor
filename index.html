怎样使用本站 How to use this site

  1. 顺序读Read in sequence. 三大模块互相引用; M2 用 M1 的报表勾稽, M3 用 M2 的估值与资本成本。 The three modules cite each other: M2 builds on M1's statement articulation, M3 builds on M2's capital costs and valuation.
  2. 三层都过Read all three layers. 直觉→定义→案例。跳过 Layer 1 直接看公式, 容易把"会算"误认为"会懂"。 Intuition → definition → case. Skipping intuition makes it easy to mistake "able to compute" for "able to understand."
  3. 动手做检验题Attempt the trap questions. 每章末 3‑5 道反直觉题; 答案折叠, 先自己想再展开。 3–5 counterintuitive prompts per chapter; the answer is folded — try before unfolding.
  4. 问 AI 助教Ask the AI tutor. 右下浮动按钮。填入你自己的 Anthropic API key 即可启用; 调用 claude-opus-4-7。 Bottom‑right floating button. Paste your own Anthropic API key to enable — backed by claude‑opus‑4‑7.
