## 思维模型清单 (每模块顶页展示)

### Module I — 会计学
1. 复式记账对称性 (任何经济事件必有双向, 类比能量守恒)
2. Accrual vs Cash 二分 (时间维度差, 类比波-粒子图像)
3. 实质重于形式 (Substance over form) — 看 control 与 risk-reward
4. 配比 (Matching) — 收入与对应费用同期
5. 谨慎性 (Conservatism) — 不预提收益, 充分预提损失
6. 历史成本 vs 公允价值张力 (客观可验 vs 决策相关)
7. 控制 (Control) — 合并范围核心标尺
8. 信号 (Signaling) — 自愿披露的博弈视角
9. 信息含量 (Information content) — 财报为何被披露

### Module II — 公司财务
1. 时间价值 (Time value of money)
2. 机会成本 (Opportunity cost)
3. 风险-收益 tradeoff (无 free lunch)
4. 价值守恒 / Value additivity (MM 的根)
5. 无套利 (No arbitrage)
6. Agency cost / 委托代理
7. Real option / 不确定性下灵活性有价
8. 资本成本 = 边际折现率
9. 信号 / 信息不对称 (Akerlof 柠檬市场)

### Module III — 证券投资学
1. 复利 (Compounding) — 指数增长反直觉
2. 能力圈 (Circle of competence)
3. Mr. Market (Graham)
4. 安全边际 (Margin of safety)
5. 护城河 (Moat) 四象限: 品牌 / 规模 / 转换成本 / 网络效应 / 特许
6. 回归均值 (Mean reversion)
7. 反身性 (Reflexivity, Soros)
8. 总回报双引擎: 增长 + 估值变化
9. 凯利公式 / 仓位与赔率

## 专家最激烈的分歧

### Module I
1. **商誉: 减值测试 vs 摊销** — IFRS/CAS 减值; FASB 2001 前摊销, 现重新讨论恢复摊销
2. **公允价值 vs 历史成本** — 2008 金融危机后 FAS 157 / IFRS 13 的顺周期之争
3. **R&D 资本化 vs 全费用化** — IFRS/CAS 部分资本化 vs US GAAP 全费用化
4. **股份支付是否为成本** — Buffett 长期主张计费; FAS 123R 之争
5. **永续债是负债还是权益** — 中国会计学会争论, 影响 D/E 与 ROE

### Module II
1. **MM 定理的现实适用性** — 简化假设打破后, 资本结构是否仍无关
2. **DCF vs 相对估值的可靠性** — Damodaran 双手都用; Buffett 偏 DCF
3. **回购 vs 派息的偏好** — MM 无关 vs signaling vs 税收 (中美税制差)
4. **β 估计方法** — 历史 β / Hamada / Bottom-up β
5. **风险溢价 (ERP) 取值** — Damodaran 隐含 ERP vs 历史 ERP vs 调查 ERP

### Module III
1. **EMH 是否成立** — Fama vs Shiller (2013 诺奖同时颁)
2. **价值 vs 成长** — Fama-French value premium vs Lynch/FAANG 拥趸
3. **主动 vs 被动** — Bogle vs Buffett (赌 S&P) vs 对冲基金
4. **技术分析是否有效** — 学术派几乎全否 vs 实务交易员
5. **A 股是否半强有效** — 国内实证: 动量/反转能否被套利消除
6. **风险 = 波动 vs 风险 = 永久性损失** — Markowitz vs Buffett

## 反直觉/陷阱题 (示意, 每章 3-5 道)

### 会计模块示例
1. 净利 +10 亿但 OCF -5 亿, 公司一定不健康吗? 列至少 3 种情景, 每种说明你接下来查附注的哪一项。
2. 间接法现金流量表中"折旧"为何作加回项? 用复式记账逻辑解释。
3. 商誉/净资产 = 50% 与 5% 两家公司, 你更担心哪家? 列两个具体附注查证动作。
4. 应收账款增速 ≫ 营收增速, 至少 3 种可能解释 + 下一步核对项。
5. 连续多年扣非净利为负但归母净利为正, 利润来自哪? 给 3 种猜想 + 验证方法。
6. 一家公司用永续债大幅置换银行借款后, ROE 上升, 这是否意味着真实经营改善?

### 公司财务模块示例
1. ROE = 20% 一定是好公司吗? DuPont 拆解出至少两种你会警惕的 ROE = 20% 情景。
2. MM 定理与现实矛盾, 把 MM 假设逐条列出, 哪条被打破对应哪类现实因素 (税盾/破产成本/信号/控制权)?
3. FCFF 折现 vs FCFE 折现理论上得到相同股权价值, 在何种条件下差异最大?
4. 高分红公司大幅降派息, 股价为何往往大跌? signaling vs free cash flow hypothesis 何时预测分歧?
5. 营运资本越多越好吗? 给两种 WC 增加但隐藏问题的情景。
6. 一家公司 EBITDA / EBIT 比值快速扩大, 可能意味着什么? (折旧政策变更 / 资本化 R&D / 减值集中)

### 投资学模块示例
1. 半强 EMH 说"已公开信息已反映"。年报披露后基本面策略仍能获利, 矛盾吗? 给两种调和方式。
2. DCF 算出茅台合理价 P*, 当前 0.7 P*。MOS 多大? 应否全仓? 用凯利公式给量化思路。
3. 茅指数/宁组合的崛起与崩塌, 用回归均值 + 反身性 交叉解释。
4. PE = 8 银行 vs PE = 80 科技股, 哪个真便宜? 列至少 3 种"PE 不可比"的来源。
5. 业绩预增 200% 公告后开盘高开 9%, 应否买入? 用半强 EMH 框架分析。
6. 一家"高 ROE + 低估值"的公司为什么市场给它低估值? 列出至少 3 种合理理由。
